
Problem : A. Mahmoud and Ehab and the even-odd game (Codeforces)

Problem link : https://codeforces.com/problemset/problem/959/A

## Solution

```C++
#include<iostream>
#include<bits/stdc++.h>
using namespace std;

int main()
{
   
   int n;
   cin>>n;

   if(n%2==0) cout<<"Mahmoud"<<endl;
   else cout<<"Ehab"<<endl;

   return 0;
}

```
